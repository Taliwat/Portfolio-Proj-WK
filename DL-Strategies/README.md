### Exploratory Data Analysis (EDA)
#### The notebooks in the 'eda' folder contain detailed exploratory analysis for each core stock, including visualizations like OBV (On-Balance Volume), Bollinger Bands, and linear trend lines.  These notebooks help understand the underlying patterns in the data before moving on to modeling.
